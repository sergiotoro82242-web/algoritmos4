# 📧 11. VALIDACIONES

## Email

```python
import re

def validar_email(email):
    patron = r'^[\w\.-]+@[\w\.-]+\.\w+$'
    return re.match(patron, email) != None
```

---

## Password

```python
def validar_password(p):
    return len(p) >= 8 and any(c.isupper() for c in p) and any(c.isdigit() for c in p)
```

# 📌 15. SUBCONJUNTOS

```python
def es_subconjunto(A, B):
    for x in A:
        if x not in B:
            return False
    return True
```

# 🧠 RESUMEN FINAL (MEMORIZAR)

* `%` → residuo
* `//` → división entera
* `set()` → elimina repetidos
* `dict.get()` → evita error
* `append()` → agrega
* `pop()` → elimina
* `len()` → tamaño
* `any()` → condición en lista

---

# 💥 TIP DE PARCIAL

Si el problema dice:

* "sin repetir" → usa **set**
* "recorrer" → usa **for**
* "validar" → usa **if**
* "dividir tiempo" → usa `//` y `%`
